## 📁 프로젝트 구조

### @Resources
- **Fonts**: Font는 Common_Fonts를 이용
- **Sound**: BGM, SFX 사운드 에셋
- **Sprites**: 게임에 사용되는 모든 이미지

### @Scenes
- 메인 씬(MainScene) 파일이 포함된 폴더
  👉 Scene 생성은 여기서 부탁드립니다.

### @Scripts
- 게임 로직 및 동작을 제어하는 스크립트 폴더
